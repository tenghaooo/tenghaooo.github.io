file descriptor

每個process有一個自己的file descriptor table，紀錄當前process的fd。

在kernel space中有一個open file table，紀錄檔案開啟的情況。
每個entry叫做open file description，其中含有offset(游標位址)，rw flag等資訊，還有指向inode的pointer，inode基本上就代表某個file本人。
